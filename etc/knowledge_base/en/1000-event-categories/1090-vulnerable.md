# Vulnerable
Category `vulnerable` points hosts with vulnerable services, having wrong configuration or services publicly available in the Internet increasing surface attack.


### 21nails;smtp
Vulnerable Exim server with 21nail vulnerability.

### 403-bypass-using-headers
Potential "403 Forbidden" bypass using (ex. X-Forwarded-For, X-Original-URL).

### 403-bypass-using-nullbyte
Potential "403 Forbidden" using nullbyte.

### adb
Publicly exposed server with Android ADB (Android Debug Bridge).

### afp
Publicly exposed server with Apple AFP (Apple Filing Protocol).

### amqp
Publicly exposed server with AMQP (Advanced Message Queueing Protocol).

### bacnet
Industrial system ICS/OT with publicly exposed service BACnet (Building Automation and Control Networks).

### bad_certificate_names
Server returns certificate for different domain.

### badsecret;basic-auth
Publicly available website with "basic-auth" having weak password.

### badwpad
Windows host with DNS misconfiguration that trying to fetch proxy configuration from "wpad.pl" server. More information at https://cert.pl/posts/2019/06/przejecie-domen-pl-zwiazanych-z-atakiem-badwpad/

### basic-auth;http
Publicly available website with "basic-auth" that is available via HTTP (password transport is done without encryption)

### certificate_authority_invalid
Certificate sign by untrusted Certificate Authority.

### cisco-smart-install
Host with publicly available Cisco Smart Install service. Using that service it is possible to modify Cisco switch configuration.

### close_domain_expiration_date

### closed_wordpress_plugin
WordPress plugin that support has ended.
