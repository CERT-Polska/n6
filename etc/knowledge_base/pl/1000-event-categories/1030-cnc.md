# Command and Control
Kategoria `cnc` (Command and Control) to hosty, które zostały zaklasyfikowane jako serwery do zarządzania botnetami. Do tych serwerów jest kierowana komunikacja z zainfekowanych urządzeń, np. wykradzione dane uwierzytelniające czy gotowość na zdalne wykonywanie polecenia z poziomu zainfekowanego komputera.
